# jscraper

## description
simple scraper for aliexpress

## requirements:
1. install via npm install
   * "express": "^4.17.1",
   * "http-proxy-middleware": "^2.0.1",
   * "morgan": "^1.10.0"
2. start js/proxy.js - it will run on 3000 port
   
3. open html/index.html write and submit your search.
